# Recommendation System Development & User Behavior Analytics

## Overview
This project implements a recommendation system for personalized learning platforms using user behavior and activity data.

## Objectives
- Analyze user learning behavior
- Recommend relevant learning modules
- Compare Content-Based Filtering and Collaborative Filtering

## Dataset Features
- User ID
- Learning Module ID
- Module Category
- Skill Name
- Time Spent
- Completion Status
- Rating
- Last Access Date

## Recommendation Techniques
### 1. Content-Based Filtering
Uses:
- Module Category
- Skill Name

### 2. Collaborative Filtering
Uses:
- User Ratings
- User Similarity

## Visualizations
- Top Accessed Learning Modules
- User Activity Distribution
- Skill Co-occurrence Heatmap
- Top Recommended Modules Dashboard
- Model Comparison Chart

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn
- Google Colab

## Results
Collaborative Filtering achieved better recommendation quality compared to Content-Based Filtering.

## Business Impact
- Personalized Learning
- Increased User Engagement
- Better Learning Outcomes
- Improved User Retention
